#YT-Dash

YT-Dash es una aplicación de escritorio para Windows que ofrece una interfaz gráfica sencilla y estable para yt-dlp, permitiendo descargar videos y audios de YouTube sin usar la consola.
Integra FFmpeg, por lo que las conversiones a MP3 y MP4 funcionan sin configuraciones adicionales.

#Características

Interfaz gráfica clara y directa para descargar contenido en MP3 o MP4.

Basada en yt-dlp, siempre actualizable y confiable.

Incluye FFmpeg incorporado.

Descargas con un clic: pegar enlace y listo.

Compatible con Windows 8, 8.1, 10 y 11.

Conversión de audio y video sin instalaciones externas.

#Requisitos

Windows 8 o superior.

Conexión a internet para las descargas.

#Instalación

Descargá la última versión desde la sección Releases del repositorio.

Descomprimí el archivo ZIP.

Ejecutá YT-Dash.exe.

No requiere instalar yt-dlp ni FFmpeg de manera independiente.

#Uso

Copiá el enlace del video o lista de reproducción.

Pegalo en el cuadro principal de YT-Dash.

Seleccioná el formato (MP3 o MP4).

Presioná Descargar.

Los archivos se guardarán en la carpeta elegida en la configuración de la app.

#Actualizaciones

YT-Dash incluye su propia copia de yt-dlp.
Revisá mejoras y nuevas versiones desde la sección de Releases del repositorio.
